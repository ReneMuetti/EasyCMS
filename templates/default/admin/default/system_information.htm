Optima -- Optima ist eine humanistische serifenlose Schrift, die von Hermann Zapf entworfen und ursprünglich 1958 von Stempel veröffentlicht wurde.
Eine aktualisierte Version wurde 2002 von Linotype unter dem Namen Optima Nova veröffentlicht.
Obwohl sie als humanistische Grotesk klassifiziert ist, genießt die Optima eher den Ruf einer Serifenschrift ohne Serifen.
Das Design zeichnet sich durch einen hohen Strichkontrast und ausgestellte Abschlüsse aus, die von römischen Inschriftenbuchstaben inspiriert sind.
Die Optima strahlt Eleganz aus und ist daher eine beliebte Schriftart für Schönheitsprodukte. 		https://www.typewolf.com
